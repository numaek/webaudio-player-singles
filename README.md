# webaudio-player-visualisation-singles
```
Some Examples for the WebAudio API:
- Frequency spectrum analyzer
- VU Meter digital
- VU Meter analog
- Oscilloscope
- Waveform-Image

Common Routing-File with 5 single Javascript-Files.
Use some or all of them.

Run under http:// because of Cross-Origin Resource Sharing (CORS policy)


For more see the WebAudio Player: 
http://www.numaek.de/Tutorial-25-WebAudio_Player.html
```
